Лицо

    Известные проблемы

  • Одной из основных проблем может быть точность трекинга и его устойчивость, особенно при изменении освещения или в условиях слабого освещения.
  • Также могут возникать проблемы с точностью распознавания различных выражений лица.

    Требования / особенности

  • Для более точного трекинга лица часто требуется хорошее освещение и четкость изображения.
  • Также может потребоваться калибровка или обучение системы для распознавания определенных выражений лица.
  • Свое ПО для создания изображения: Это полностью открытое для использование ПО
Дополнительная информация

    Фейс-группа

  • По умолчанию, когда в иерархию добавляется новая группа лиц, отображаются Исходная точка, Левый глаз, Правый глаз, Центр рта и подбородок.
  • Дополнительные точки прикрепления можно добавить в иерархию, щелкнув правой кнопкой мыши на узле Группы лиц и выбрав Создать > Точка прикрепления лица

    Фейсовые события

  • Группа лиц генерирует события на основе различных движений лица. Их можно активировать с помощью действий или прослушивания события в сценарии.
  • По умолчанию содержимое в группе лиц не маскируется, что означает, что 3D-модель будет отображаться полностью поверх лица пользователя.
