Изображение

    Известные проблемы

  • Ограничения по количеству изображений, которые можно отслеживать одновременно
  • Ограничения по размеру и качеству изображений.

    Требования / особенности

  • Нужно изображение хорошего качества
  • Для трекинга необходимо иметь четкие изображения с яркими контурами, которые можно использовать для идентификации.
  • Свое ПО для создания изображения: Это полностью открытое для использование ПО
Дополнительная информация
  • В данном видеоматериале можно ознакомиться с тем как выглядит программное обеспечение и его особенности, а также как им пользоваться
  • После выбора отслеживания изображений будет предложено загрузить целевое изображение. Ваше целевое изображение будет подготовлено и после загрузки появится на вашем холсте.
  • Отслеживание изображений позволяет обнаруживать и преобразовывать цифровой контент в плоское или вертикальное изображение
  • Это отлично подходит для создания контента, дополняемого печатными материалами, такими как визитные карточки, плакаты и страницы журналов.
