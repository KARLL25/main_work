Изображение

    Известные проблемы

  • Работает на всех телефонах с webgl и webrtc
  • Отслеживание изображений требует больше ресурсов процессора
  • На Android / Chrome можно столкнуться с проблемами из-за возможного аппаратного ограничения устройства.

    Требования / особенности

  • Нужно изображение хорошего качества
  • Отслеживание изображений требует больше ресурсов процессора
